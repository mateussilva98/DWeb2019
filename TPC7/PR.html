    • Abstract:

    Este trabalho consistiu essencialmente em duas tarefas:

    1ª Tarefa: Criar três queries em MongoDB que permitam obter o pedido.

    2ª Tarefa: Criar um servidor em Node.js usando o Express que disponibilize e permita gerir os documentos referentes a uma base de dados de filmes guardados no MongoBD.

    De forma a ser possível gerir esta informação, há varias operações que devem estar implementadas, nomeadamente os serviços CRUD (Create Read Update Delete). São estas operações a listagem de todos os filmes, a consulta de um filme, a adição de um novo filme, a remoção de filmes e a atualização de filmes já existentes.

    Como forma de simplificação e para não sobrecarregar o servidor, na listagem de todos os filmes limitei o número de filmes que são apresentados. Ainda tentei implementar um sistema de paginação, mas não tive tempo para tal. Para além disso, não é possível adicionar e remover atore e generos nos filmes.

    Deliverables:

    1. PR.xml - XML do Project Record
    2. PR.html - HTML do Project Record

    4. config - pasta que contém um ficheiro com as configs e um ficheiro que trata da ligação à base de dados
    5. controllers - pasta que contém o controller dos filmes
    6. models - pasta que contém o model dos filmes
    7. javascripts - pasta que contém o ficheiro operacoes.js que contém funções de auxilio aos pedidos delete e put
    8. stylesheets - pasta de contém o stylesheet
    9. routes - pasta de contém o toute de filmes
    10. views - pasta de contém as views
