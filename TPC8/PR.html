    • Abstract:

    Este trabalho consistiu, essencialmente, na construção de uma API de dados para a base de dados de Prémios Nobel.

    A API de dados que foi necessário desenvolver corresponde a um exercício que pode ser encontrado aqui.

    Para a construção desta aplicação WEB foram fundamentais as ferramentas node Express, mongoose e axios.

    Deliverables:

    1. PR.xml - XML do Project Record
    2. PR.html - HTML do Project Record
    3. config - pasta que contém um ficheiro com as configs e um ficheiro que trata da ligação à base de dados
    4. controllers - pasta que contém o controller dos premios
    5. models - pasta que contém o model dos premios
    6. stylesheets - pasta de contém o stylesheet
    7. routes - pasta de contém o toute de premios
    8. views - pasta de contém as views
    9. server.js - ficheiro do server
    10. package.json - ficheiro com a forma como podemos correr o server e onde são referidas as suas dependências
