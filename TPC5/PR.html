Servidor para listar tarefas
Supervisor José Carlos Ramalho
Data Início 19-10-2019
Data Final 21-10-2019
  • A81952 - Mateus Ferreira da Silva - a81952@alunos.uminho.pt

    • Abstract:

    Foi proposto que fosse feito um servidor para gestão de tarefas. Este servidor deve responder a um pedido GET ('/' ou '/index) sendo apresentada uma página com uma lista de tarefas registadas anteriormente e onde é possível acrescentar novas tarefas através de uma pedido POST.

    Para realizar o trabalho foi necessário criar três ficheiros pug. O layout.pug que é igual ao desenvolvido na aula e vai ser extendido pelos dois outros ficheiros pug. O fiheiro index.pug é responsável por apresentar a página principal. Por fim, o erro.pug é o template a ser mostrado quando é feito um pedido não suportado pelo servidor.

    O servidor foi desenvolvido em Node.js e está no ficheiro gestao-tarefas.js e deve processar os pedidos e responder-lhes da forma mais adequada.

    É importante também referir que é usado um ficheiro lista_tarefa.jsons onde são guardadas as tarefas registadas.

    Deliverables:

    1. w3.css - stylesheet
    2. layout.pug
    3. erro.pug
    4. index.pug
    5. gestao-tarefas.js - servidor
    6. lista_tarefas.json
    7. PR.xml - XML do Project Record
    8. PR.html - HTML do Project Record
    9. files - pasta com o favicon.ico
