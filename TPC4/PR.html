Transformação de XML com XSL e servidor de páginas em Node.js
Supervisor José Carlos Ramalho
Data Início 11-10-2019
Data Final 12-10-2019
  • A81952 - Mateus Ferreira da Silva - a81952@alunos.uminho.pt

    • Abstract:

    Este trabalho pode ser dividido em duas tarefas:

    - Criar uma stylesheet (XSL) para transformar o ficheiro de arqueossítios (XML), disponivel aqui, em vários ficheiros XML, cada um com os distintos arqueossítios. Após isso é usada uma stylesheet para transformar o XML em HTML.

    - Criar um um servidor de ficheiros, escrito em JavaScript e usando Node.js que disponibilize os ficheiros XML criados na tarefa anterior. Estes ficheiros podem ser acedidos através da especificação na query string do URL.

    Deliverables:

    1. arq2html.xsl - stylesheet para transformação dos XML dos arqueossítios em HTML
    2. xml-split.xsl - stylesheet para criar os ficheiros XML individuais
    3. server.js - servidor
    4. PR.xml - XML do Project Record
    5. files - pasta com os ficheiros XML
    6. PR.html - HTML do Project Record
